# PopX - Mobile App Interface

A pixel-perfect React application built as a qualifier task for an interview assignment. The mobile app interface is centered on the webpage with seamless navigation between all pages.

## 🚀 Live Demo

[View Live on Vercel](https://popx-app.vercel.app)

## ✨ Features

- **Pixel-Perfect UI**: Mobile interface (375×812px) precisely centered on the webpage matching the Adobe XD design
- **Seamless Navigation**: Utilizes `react-router-dom` for smooth transitions between pages
- **Dynamic Profile State**: Passes user data from Login/Signup screens to Profile using React state
- **Custom UI Components**:
  - Modular `InputField` components with floating labels using `<fieldset>`
  - Custom-styled radio buttons for the "Agency" selection
- **Clean Code**: Well-formatted, organized component architecture

## 📱 Screens

1. **Welcome** - Landing page with decorative circles and CTA buttons
2. **Login** - Sign in form with email and password
3. **Create Account** - Registration form with all required fields
4. **Account Settings** - User profile display with avatar and bio

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS
- **Routing**: React Router DOM
- **Hosting**: Vercel

## 💻 Local Development

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/popx-app.git

# Navigate to project directory
cd popx-app

# Install dependencies
npm install

# Start the development server
npm run dev
```
